# CostAnalyzer 💰

多柜项目成本计算与报价优化系统，基于 Streamlit 的网页端应用。

## 功能特性

- **多柜项目配置** — 支持多台柜子独立计算和项目汇总
- **元器件管理** — 手动添加/删除，支持 CSV/Excel 批量导入，复制柜子
- **成本分析报告** — 自动计算铜排、电缆、箱体、辅材等各项费用
- **铜价自动获取** — 从新浪财经 API 获取沪铜实时价格
- **SQLite 价格库** — 2212 条产品数据，支持多用户并发编辑（WAL 模式）
- **模糊搜索** — 按型号或名称快速查找元器件
- **类型自动识别** — 从价格库自动匹配元器件类型（塑壳/框架断路器等）
- **计算日志** — 自动保存最近 1500 次计算的配置和结果
- **操作日志** — 记录价格库变更历史

## 页面结构（5 个 Tab）

1. **项目配置** — 柜子数量、参数设置、元器件清单管理
2. **元器件管理** — 添加/删除元器件，批量导入
3. **成本分析报告** — 各柜成本明细与项目汇总
4. **公式说明** — 计算公式与参数说明
5. **价格库管理** — 网页端浏览、搜索、编辑价格数据
