# CostAnalyzer 💰

成本计算与报价优化系统。

## 功能

- 成本分析与计算
- 报价优化建议
- 价格数据库管理
- 数据导入（支持 Excel）

## 文件结构

```
cost-analyzer/
├── app.py          # 主程序
├── price_db.json   # 价格数据库
├── data/           # 数据目录
├── templates/      # 模板目录
├── start.sh        # 启动脚本
├── 成本.xls        # 成本数据源
└── 成本.xlsx       # 成本数据源
```

## 使用方法

```bash
bash start.sh
```
