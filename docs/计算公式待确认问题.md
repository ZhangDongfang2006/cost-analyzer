## 问题1：200A/225A断路器铜排/电缆归属

**现状：** 代码判断 `current >= 250` 走铜排，`else` 走电缆。但200A和225A不在电缆截面积对照表（63/80/100/125/160）中，导致200A/225A既不走铜排也不走电缆，费用为0。

**选项：**
- A. ≥200A统一走铜排（200A/225A也走铜排）
- B. ≥250A走铜排不变，200A/225A加到电缆对照表（200A→95mm², 225A→120mm²）
- C. 保持现状，实际项目中不使用200A/225A

**影响：** 如果有200A/225A断路器，成本会少算。

## 问题5：零线N截面积计算方式

**现状：** 零线N截面积固定为主母线截面积的1/2（S/2）。

**国标参考（GB/T 7251）：**
| 相线截面积S(mm²) | 中性线最小截面积 |
|---|---|
| S≤16 | S |
| 16<S≤25 | 16 |
| 25<S≤35 | 25 |
| 35<S≤400 | S/2 |
| S>400 | S/2 |

**疑问：** 当前S/2在S>35时基本符合国标，但S≤35时可能有偏差。不过实际项目中铜排规格最小也是15×2=30mm²，影响不大。是否需要按国标修正？
