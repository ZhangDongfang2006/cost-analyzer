## 1. 铜排基本知识

### 1.1 铜排命名规则
- **TMY** = 铜母线（硬铜排），T（铜）、M（母线）、Y（硬态）
- 规格表示：**宽 × 厚**，单位 mm，如 TMY-60×6 表示宽60mm、厚6mm
- 截面积 = 宽 × 厚（mm²）

### 1.2 常用铜排规格

| 规格 (宽×厚 mm) | 截面积 (mm²) | 每米重量 (kg/m) |
|---|---|---|
| 30×3 | 90 | 0.80 |
| 30×4 | 120 | 1.07 |
| 40×4 | 160 | 1.42 |
| 50×5 | 250 | 2.22 |
| 60×6 | 360 | 3.20 |
| 60×8 | 480 | 4.27 |
| 80×6 | 480 | 4.27 |
| 80×8 | 640 | 5.69 |
| 80×10 | 800 | 7.11 |

> 铜的密度 ≈ 8.89 g/cm³，每米重量 = 截面积(mm²) × 0.00889 kg

## 2. 铜排载流量计算

### 2.1 基本公式

**最简估算公式：**
```
载流量 I = 截面积 S × 电流密度 J
```

其中电流密度 J 的选取取决于散热条件、温升要求等。

### 2.3 温度修正系数

不同环境温度下的载流量修正系数：

| 环境温度 (℃) | 修正系数 |
|---|---|
| 25 | 1.12 |
| 30 | 1.07 |
| 35 | 1.00 |
| 40 | 0.93 |
| 45 | 0.85 |
| 50 | 0.76 |

公式：`I_actual = I_35℃ × K_temp`

温升修正（不同最大允许温升）：
- 25K（正常）：系数 = 1.00
- 35K（提高）：系数 ≈ 1.20
- 50K（极限）：系数 ≈ 1.40

### 2.4 排列方式影响

| 排列方式 | 说明 | 载流量系数 |
|---|---|---|
| **立放（竖放）** | 宽面朝向散热方向，散热好 | 1.00（基准） |
| **平放（横放）** | 窄面朝上，散热差 | 0.90~0.95 |
| 两片叠放 | 第二片散热受限 | 第1片×1.00，第2片×0.95 |
| 三片叠放 | | 第1片×1.00，第2片×0.90，第3片×0.85 |
| 四片叠放 | | 第1片×1.00，第2片×0.85，第3片×0.75，第4片×0.70 |

**多片并联总载流量 = 各片修正后载流量之和**

### 2.5 间距影响
- 铜排间距 ≥ 1倍厚度：正常散热
- 间距 < 1倍厚度：需要降容 5%~10%
- 建议间距：≥ 1.5倍厚度

## 3. 中国国家标准要求

### 3.1 GB/T 7251.1-2013 《低压成套开关设备和控制设备》

**关键要求：**
- 母线（铜排）的选择应满足正常工作条件下的载流能力
- 温升限值：外部可触及部分 ≤ 30K，内部可触及部分 ≤ 40K，绝缘部件 ≤ 70K
- 母线连接处温升不应超过导体本身温升的限值
- 需考虑短路电流下的动稳定和热稳定

### 3.2 GB/T 5585.1-2005 《电工用铜、铝及其合金母线 第1部分：铜和铜合金母线》

**关键要求：**
- 规定了铜母线的型号（TMY-硬铜母线、TMR-软铜母线）
- 标称尺寸及允许偏差
- 电阻率要求：20℃时电阻率 ≤ 0.01777 Ω·mm²/m
- 抗拉强度：TMY ≥ 245 MPa，TMR ≥ 205 MPa

### 3.3 GB 50054-2011 《低压配电设计规范》

**关键要求：**
- 第3.2.2条：导体截面应满足长期允许载流量不小于计算电流
- 第3.2.9条：导体应满足短路条件下的热稳定要求
- 绝缘导体温度限值：PVC ≤ 70℃，XLPE ≤ 90℃

### 3.4 DL/T 375-2010 《户外配电箱通用技术条件》

- 母线截面选择应按计算电流确定
- 考虑环境温度修正

### 3.5 JGJ 16-2008 《民用建筑电气设计规范》

- 母线载流量应考虑环境温度修正系数
- 封闭式母线槽按厂家提供的额定电流选择

### 4.3 成本优化建议

铜排成本 = 截面积 × 长度 × 铜单价 + 加工费用

- 铜价约 70~80 元/kg（2026年参考价）
- 加工费用（折弯、钻孔、搪锡）：约 5~15 元/m
- 单片铜排比多片并联更经济（加工简单、连接件少）
- 但大截面积铜排散热差，有时双片比超厚单片更优

### 5.1 简化选择算法

```python
    {'spec': '120×10', 'width': 120, 'thickness': 10, 'area_mm2': 1200, 'current': 2330},
], key=lambda x: x['current'])

def select_copper_busbar(current: float) -> dict:
    """根据电流选择最小满足要求的铜排规格"""
    for spec in COPPER_BUSBAR_TABLE:
        if spec['current'] >= current:
            return spec
    return COPPER_BUSBAR_TABLE[-1]
```

### 5.2 成本估算公式

```python
def estimate_busbar_cost(current: float, length: float, 
                         copper_price_per_kg: float = 75,
                         process_cost_per_m: float = 10) -> float:
    """
    估算铜排成本
    current: 额定电流 (A)
    length: 总长度 (m)
    copper_price_per_kg: 铜价 (元/kg)
    process_cost_per_m: 加工费 (元/m)
    """
    selection = select_copper_busbar(current)
    area = selection["area"]
    count = selection.get("count", 1)
    
    # 重量 = 截面积 × 长度 × 密度(8.89 g/cm³)
    weight_per_meter = area * 0.00889  # kg/m
    total_weight = weight_per_meter * length * count
    
    # 材料费
    material_cost = total_weight * copper_price_per_kg
    
    # 加工费
    process_cost = length * count * process_cost_per_m
    
    return material_cost + process_cost
```

## 6. 总结与建议

### 6.1 核心结论

1. **电流密度取值**：对于配电柜成本估算，推荐取 **1.2~1.5 A/mm²** 作为默认值
2. **截面积越大电流密度越低**：小截面(90mm²)可达2.5 A/mm²，大截面(1200mm²)仅约1.3 A/mm²
3. **优先选单片**：同电流下单片铜排比多片并联成本更低（加工简单、连接件少）
4. **铜价波动大**：建议铜排成本与铜价联动计算

### 6.2 成本计算系统建议

1. **建立铜排载流量对照表**（如上第5.3节），作为系统内置数据
2. **输入参数**：额定电流、环境温度、铜排长度
3. **输出**：推荐铜排规格、数量、重量、材料费、加工费、总成本
4. **预留铜价参数**：支持实时铜价输入
5. **简化模式**：提供"按电流密度快速估算"和"按标准载流量精确计算"两种模式

### 6.3 注意事项

- 本报告数据基于35℃环境温度、温升25K条件
- 实际工程中需考虑短路电流热稳定校验
- 柜内通风条件会影响实际载流量
- 建议在系统中标明"仅供成本估算参考，实际设计需由专业电气工程师确认"
