# 成本计算系统使用指南

> 配电设备成本计算系统，用于计算配电柜的成本和报价。

---

## 第一部分：网络连接（Tailscale）

### 一、下载安装

**Windows 电脑：**
1. 浏览器打开 https://tailscale.com/download/windows
2. 点击 Download for Windows
3. 运行安装程序，按提示完成安装

**手机（iPhone/Android）：**
- iPhone：App Store 搜索 Tailscale → 安装
- Android：应用商店搜索 Tailscale → 安装

### 二、登录

1. 打开 Tailscale 应用
2. 点击 Log in
3. 使用分配给你的账号登录
4. 登录后显示等待页面是正常的，请通知CC审批

### 三、审批通过后

CC审批通过后，Tailscale状态会自动变为 Connected。

此时浏览器输入以下地址即可访问成本计算系统：

```
http://100.102.56.99:8502/
```

建议收藏此地址，方便下次使用。

### 四、日常使用

- Tailscale 默认开机自启动，无需手动操作
- 如果状态显示 Not connected，点击 Connect 重新连接即可

## 第二部分：成本计算系统

### 五、访问地址

```
http://100.102.56.99:8502/
```

必须先安装并登录 Tailscale（见第一部分），否则无法访问。

### 六、系统界面

系统共有 5 个页面：

| 页面 | 功能 |
|------|------|
| 元器件管理 | 添加柜子、添加元器件、生成报价 |
| 批量导入 | 从Excel文件或文本粘贴批量导入元器件 |
| 成本分析报告 | 查看成本明细、铜排/电缆费用、利润、报价 |
| 计算公式说明 | 查看所有计算规则和公式 |
| 价格库管理 | 搜索、编辑、导入产品价格数据 |

### 七、基本使用流程

#### 步骤 1：创建柜子

1. 点击 元器件管理 页面
2. 点击 添加柜子
3. 输入柜子名称（如"出线柜1"）
4. 选择柜子类型（出线柜/进线总柜/联络柜/电容柜/切换柜）
5. 输入柜宽（如 0.8）

#### 步骤 2：添加元器件

1. 选择要添加元器件的柜子
2. 输入元器件型号（如 XT1N160 TMD 100 3P FF）
3. 系统自动匹配类型、品牌、单价
4. 输入数量
5. 点击 添加

如果类型显示为空并有红色提示，说明系统未识别该型号，请手动选择类型。

#### 步骤 3：生成成本报告

1. 在元器件管理页面点击 计算成本分析报告
2. 系统自动计算并显示报告

#### 步骤 4：查看和导出报告

报告包含每台柜子的明细：
- 元器件清单（名称、型号、数量、单价、金额）
- 铜排费用
- 电缆费用
- 辅助材料
- 箱体费用
- 成套费（利润）

底部显示项目汇总和含税报价，可点击 下载CSV 导出。

### 八、批量导入（可选）

1. 点击 批量导入 页面
2. 选择目标柜子
3. 上传Excel文件或在文本框中粘贴元器件信息
4. 点击 导入

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| 无法访问网页 | 确认Tailscale已登录且状态为Connected |
| 登录后一直等待 | 已通知CC，等待审批即可 |
| 审批后无法访问网页 | 确认Tailscale显示Connected |
| 页面打不开 | 确认地址是 http://100.102.56.99:8502/（http不是https） |
| Tailscale断开 | 打开Tailscale点击Connect重连 |
| 元器件类型为空 | 手动选择正确类型 |
| 辅材未匹配 | 系统会提示，可手动输入价格 |
| 报告为空 | 确认已添加元器件并点击"计算成本分析报告" |
