# Changelog

## v1.0 (2026-03-31)

### 核心功能
- 多柜项目成本计算系统，支持多台柜子独立计算和项目汇总
- SQLite 价格库（2212 条产品），支持多用户并发编辑（WAL 模式）
- 网页端价格库管理（浏览、搜索、编辑）
- CSV/Excel 批量导入元器件
- 新浪财经 API 自动获取沪铜实时价格
- 计算日志（最近 1500 次），操作日志记录价格库变更
